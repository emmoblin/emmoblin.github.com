串口工具ckermit


默认配置文件

/etc/kermit/kermrc
一般配置一个home下的自己的配置文件就ok了。
~/.mykermrc


set line /dev/ttyS0
set speed 9600
set carrier-watch off
set handshake none
set flow-control none
robust
set file type bin
set file name lit
set rec pack 1000
set send pack 1000
set window 5

C-\ 切换到命令行
c 打开链接
